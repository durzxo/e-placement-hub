## ğŸš€ Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Aradhyya/APSIT-Placement-Portal-.git
   cd e-placement-hub
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   ```
   http://localhost:5173/
   ```

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```
